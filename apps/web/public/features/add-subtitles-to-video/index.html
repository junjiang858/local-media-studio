Subtitle FAQ.

Are subtitles generated automatically?

No. The first MVP supports manual subtitles, not AI caption generation.

Can I edit subtitle timing?

Yes. Subtitle cues include editable start and end times.

Is this for long-form video production?

No. Obscura is optimized for short single-video preparation in v1.
