Background remover FAQ.

Does background removal work offline?

The app can process locally, but the model/runtime assets must be available first.

Is the result editable?

The result is added as a separate local asset, so it can be inspected and exported.

Is manual mask refinement included?

No. Manual erase and restore refinement is deferred beyond the first MVP.
