Video trimmer FAQ.

Are videos uploaded to a server?

No. Obscura v1 has no backend upload endpoint for user media.

Which videos work best?

Short videos in browser-friendly formats are the best fit for the first MVP.

Can I convert formats?

Yes. Obscura supports local video format export paths where browser processing allows it.
