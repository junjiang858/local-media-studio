Image conversion FAQ.

Which formats are available?

Obscura supports browser-native formats plus selected local encoders where feasible.

Can I convert after editing?

Yes. You can crop, adjust, annotate, watermark, then export in the chosen format.

Does conversion upload the file?

No. The first version keeps user media local to the browser session.
