No-upload background removal

Remove a background without sending the image away.

Obscura gives creators a practical way to cut out image backgrounds inside the browser. The selected source image stays in the local editing workflow instead of being uploaded to a backend media service.

Background removal FAQ.

Does Obscura upload the source image?

No. Obscura v1 has no backend media upload endpoint for user files.

Can I edit after removing the background?

Yes. You can continue with crop, resize, annotation, watermarking, and export.

Is edge quality always perfect?

No automatic cutout is perfect for every image. Fine hair, glass, and motion blur can need manual review.
