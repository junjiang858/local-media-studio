在线抠图 FAQ。

模型资源来自哪里？

应用可能下载静态模型或运行时资源，这是应用资源加载，不是上传你的图片。

抠图质量适合什么场景？

适合日常创作和轻量素材处理，复杂发丝、透明物体和反光边缘可能需要后续人工修正。

会覆盖原图吗？

不会。生成结果作为新的会话素材出现，原始图片仍然保留在媒体库中。
