浏览器剪辑视频 FAQ。

适合多大的视频？

v1 面向短视频和轻量素材。大文件和长视频可能因为浏览器内存或编码限制而变慢。

会修改原视频吗？

不会。Obscura 会生成派生预览或导出结果，原始素材仍然保留。

是否支持专业多轨剪辑？

不支持。Obscura v1 的目标是快速处理单个短视频，而不是替代完整剪辑软件。
