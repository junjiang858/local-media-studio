隐私边界清晰。

Obscura 不是云端素材库，也不是协作平台。它的重点是把常见媒体处理留在浏览器工作流里。

没有后端上传

v1 不提供上传用户图片或视频到后端进行处理的接口。

没有账号系统

首版不包含登录、团队协作、云同步或远程媒体库。

应用资源可下载

模型、WASM 或脚本资源可能加载到浏览器，但用户媒体不因此上传。

本地优先 FAQ。

文件名会作为分析事件发送吗？

不会。项目文档明确禁止把文件名、字幕、媒体内容、路径或派生内容放进分析数据。

导出结果会自动保存到云端吗？

不会。导出结果由用户在浏览器中下载，v1 没有远程持久化。

未来会加入云端处理吗？

除非先更新项目范围并得到明确确认，否则 v1 不会加入云端媒体处理。
